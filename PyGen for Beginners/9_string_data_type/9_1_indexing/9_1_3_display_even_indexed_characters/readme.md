# Even Indexed Characters Display

## Description 📝

This program reads an input string from the user and displays the characters located at even indices (0, 2, 4, ...) in the string. Each character is displayed on a new line.

## Purpose 🎯

The purpose of this program is to extract and display the characters located at even indices in the input string, helping users practice string manipulation and indexing in Python.

## How It Works 🔍

1. The program takes an input string from the user.
2. It iterates through the string, selecting only the characters at even indices (i.e., index 0, 2, 4, etc.).
3. For each character at an even index, it is printed on a new line.

## Output 📜

The program prints each character located at even indices in the input string, each on a new line.

## Usage 📦

1. Run the Python script.
2. Input a string when prompted.
3. The program will display the characters at even indices, each on a new line.

### Example Usage:

1. Run the script.
2. Input: `python`
3. Output: `p t o`

## Conclusion 🚀

This simple program demonstrates string indexing and is useful for beginners learning to manipulate and work with strings in Python.
